== FAQ ==

= How to install import eventbrite event free version? =

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins & Add New.
2. Type "Import Eventbrite Events" into the Search input and click the "Search" button.
3. Locate the "Import Eventbrite Events" in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

It can also be installed manually.

1. Download the "Import Eventbrite Events" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Import Eventbrite Events" in the list and click the "Activate" link.
